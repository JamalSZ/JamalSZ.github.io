GitHub Pages 404 Error Fix

If you're seeing a 404 error for your GitHub Pages site (jayzs.github.io), follow these steps to resolve the issue:

1

Check Repository Name

Ensure your repository is named exactly jayzs.github.io (case-sensitive).

Repository name is incorrect

Rename your repository to jayzs.github.io

# To rename your repository:
# 1. Go to your repository on GitHub
# 2. Click on "Settings"
# 3. Under "Repository name", change it to "jayzs.github.io"
# 4. Click "Rename"
2

Check Branch Settings

GitHub Pages needs to be configured to use the correct branch.

Wrong branch selected for GitHub Pages

Set GitHub Pages to use the main branch:

# Configuration steps:
# 1. Go to your repository on GitHub
# 2. Click on "Settings"
# 3. Scroll down to "GitHub Pages" section
# 4. Under "Source", select "main" branch (or "master")
# 5. If you're using a docs folder, select "/docs"
# 6. Click "Save"
3

Check for index.html

Your repository must contain an index.html file in the root or selected folder.

Missing index.html file

Add an index.html file to your repository:

# To add index.html:
# 1. Go to your repository on GitHub
# 2. Click "Add file" → "Create new file"
# 3. Name the file "index.html"
# 4. Copy the HTML code from this page
# 5. Scroll down and click "Commit new file"
4

Wait for Deployment

GitHub Pages can take a few minutes to deploy your site after changes.

Site not deployed yet

Wait a few minutes and refresh your page:

# After making changes:
# 1. Wait 1-5 minutes for GitHub to process your changes
# 2. Refresh your browser at https://jayzs.github.io
# 3. If still not working, check the "Settings" → "Pages" section
# for any error messages
5

Check for Build Errors

If you're using Jekyll or other static site generators, check for build errors.

Build errors in GitHub Actions

Check build status:

# To check for build errors:
# 1. Go to your repository on GitHub
# 2. Click on "Actions" tab
# 3. Look for any failed workflows
# 4. If using a custom static generator, ensure it's supported by GitHub Pages

Still having issues?

Visit the GitHub Pages documentation for more help.
